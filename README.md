# QR Code Generator Project

A modern, single-page QR Code Generator that allows users to:

Generate QR codes from any text or URL

Download the generated QR code as a PNG image

Copy the QR code image instantly

Copy the entered text or URL

Use a clean and intuitive interface


## 📝 What Does It Do?

The app allows users to instantly create QR codes from any text or link.

Text or URL Input: Users can enter any text, website link, or information into the input field.

Generate QR Code Button: When clicked, the application generates a QR code based on the entered content.

QR Code Display: The generated QR code appears instantly on the screen.

Download PNG: Users can download the QR code as a PNG image.

Copy PNG: Allows users to copy the generated QR code image to the clipboard.

Copy Text: Users can quickly copy the entered text or URL.

Modern UI Design: A clean card-based interface with gradient background, rounded corners, and soft shadows.

The interface is minimal, responsive, and designed for quick and easy QR code creation.


## ⚙️ How Does It Work?

Single-page structure: Built using HTML, CSS, and JavaScript.

QR code generation: JavaScript dynamically generates a QR code from the entered text or URL.

Event-driven interaction: Clicking the Generate QR Code button triggers the QR generation function.

Clipboard API: Copy buttons allow users to copy the QR code image or text to the clipboard.

Dynamic rendering: The QR code appears instantly without refreshing the page.

Download feature: Users can download the generated QR code as a PNG file.

Lightweight implementation: No heavy frameworks — pure JavaScript and DOM manipulation.
